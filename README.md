### 🛠️ What I Build

I build **ClawOS** — an agent-native operating system that runs entirely offline with no API keys. Local AI that actually works.

- **AI agents** that run on your hardware, not someone else's cloud
- **Safety-first tooling** — commands that default to "no" on dangerous operations
- **Offline pipelines** — from script to published video, zero internet required
- **3D visualizations** — because dashboards should look like something, not just show something
